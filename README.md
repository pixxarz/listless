# 📋 ระบบรายงานนักเรียนเวลาเรียนไม่ถึงร้อยละ 70

ระบบสำหรับ **โรงเรียนชานุมานวิทยาคม** ให้ครูรายงานรายชื่อนักเรียนที่มีเวลาเรียนไม่ถึงร้อยละ 70 บันทึกลง Google ชีต แล้วพิมพ์เป็นเอกสารราชการ A4 ได้ พร้อมหน้าแดชบอร์ดสรุปข้อมูลทั้งหมด

เว็บล้วน (HTML/CSS/JavaScript) ไม่ต้องติดตั้งอะไร · หลังบ้านใช้ Google Apps Script + Google ชีต · โฮสต์บน Netlify

---

## ✨ ความสามารถ

### หน้ากรอกฟอร์ม — `index.html` (สำหรับครู)
- กรอกข้อมูลรายวิชา + รายชื่อนักเรียนที่เวลาเรียนไม่ถึงเกณฑ์ (คำนวณคาบขาด + ร้อยละให้อัตโนมัติ)
- เตือนทันทีเมื่อกรอกเลขที่ซ้ำ และกันบันทึกถ้านักเรียนเวลาเรียนถึงร้อยละ 70 (ไม่เข้าเกณฑ์)
- บันทึก 2 ขั้น (ตรวจทาน → ยืนยัน) กันบันทึกพลาด
- พิมพ์เอกสารราชการ A4 (ปุ่มเดียว "พิมพ์ / บันทึก PDF")
- ใช้ได้ทั้งคอมและมือถือ

### หน้าแดชบอร์ดรายงาน — `report.html` (สำหรับผู้ดูแล)
- เข้าด้วยรหัสผ่าน (ตรวจที่ฝั่งเซิร์ฟเวอร์)
- การ์ดสรุปตัวเลข (KPI) + การ์ดเจาะลึก 4 แบบ: รายชื่อนักเรียน / ใบรายงาน / ครู / รายวิชา
- ตารางหลักค้นหา-กรอง-เรียงได้ (เรียงซ้อนหลายชั้น + ตรึงคอลัม)
- ดาวน์โหลด CSV
- พิมพ์เอกสารได้ 2 แบบ:
  - **ใบรายงานครูเดี่ยว** (แยกตามครู/วิชา)
  - **บัญชีรายชื่อรวม** นักเรียนเวลาเรียนไม่ถึงร้อยละ 70 ทุกวิชา จัดกลุ่มตามระดับชั้น
- เช็ครายงานใหม่อัตโนมัติแบบเรียลไทม์

## 🚀 การใช้งาน

**ครู (กรอกข้อมูล):** เปิดหน้าฟอร์ม → กรอกข้อมูลวิชา + รายชื่อนักเรียน → กดบันทึก (ตรวจทาน → ยืนยัน) → กดพิมพ์เพื่อได้เอกสาร A4

**ผู้ดูแล (ดูรายงาน):** เปิดหน้ารายงาน → ใส่รหัสผ่าน → ดู/กรอง/พิมพ์/ดาวน์โหลดข้อมูล

## ⚙️ การติดตั้ง / Deploy

### หน้าเว็บ (Netlify)
ผูกกับ GitHub repo — ทุกครั้งที่ push ขึ้น branch `main` Netlify จะ deploy อัตโนมัติ ไม่ต้องตั้งค่าเพิ่ม

### หลังบ้าน (Google Apps Script)
1. เปิดโปรเจกต์ Apps Script ที่ผูกกับ Google ชีต → วางโค้ดจาก `Code.gs`
2. แก้ไขโค้ดแล้วต้อง **Deploy เวอร์ชันใหม่** ทุกครั้ง (Manage deployments → New version) — ลิงก์ (`SHEET_URL`) จะไม่เปลี่ยน
3. ข้อมูลถูกเก็บในชีตชื่อ **"รายงาน"** (สร้างหัวตารางให้อัตโนมัติถ้ายังไม่มี)

### ตั้งรหัสผ่านหน้ารายงาน
แนะนำให้ตั้งแบบปลอดภัยผ่าน Apps Script:
Project Settings (รูปเฟือง) → Script Properties → Add → ตั้งชื่อ `REPORT_PASSWORD` แล้วใส่รหัสที่ต้องการ
(ถ้าไม่ตั้ง จะใช้ค่าเริ่มต้นในโค้ด ซึ่งควรเปลี่ยนก่อนใช้งานจริง)

## 📝 หมายเหตุ
- เกณฑ์เวลาเรียนกำหนดที่ตัวแปร `THRESHOLD` (ค่าปัจจุบัน 70)
- การบันทึกกันข้อมูลซ้ำ/หายด้วยรหัสประจำใบ (`submission_id`) + ระบบล็อกกันบันทึกชนกัน + ยืนยันว่าข้อมูลเข้าชีตจริง
- วันที่ในชีตเป็นเวลาสากล (UTC) เว็บปรับเป็นเวลาไทย (+7) ก่อนแสดงเสมอ
